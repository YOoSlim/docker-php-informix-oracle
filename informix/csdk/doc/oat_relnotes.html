Description

OAT is a web application for administering database servers and analyzing database-server performance. You can administer multiple database server instances from a single OAT installation on a web server. You can access the web server through any browser to administer all your database servers.

OAT includes these plug-ins:
  • The IBM Informix Health Advisor Plug-in for OpenAdmin Tool (OAT)
  • The IBM Informix Replication Plug-in for OpenAdmin Tool (OAT)
  • The IBM Informix Schema Manager Plug-in for OpenAdmin Tool (OAT)
  • The IBM Informix TimeSeries Plug-in for OpenAdmin Tool (OAT)
  • The IBM Informix JSON Plug-in for OpenAdmin Tool (OAT)
You can create additional plug-ins for OAT to add the functions that you need.

For a complete list of what's new in this release, go to http://www.ibm.com/support/knowledgecenter/SSGU8G_12.1.0/com.ibm.po.doc/new_features_ce.htm.

Installing OAT

OAT is installed by default during a typical installation of the IBM Informix Client Software Development Kit (Client SDK) or when you install the IBM InformixIBM Informix software bundle and select Client SDK.

For installation instructions, see this topic in IBM Knowledge Center: http://www.ibm.com/support/knowledgecenter/SSGU8G_12.1.0/com.ibm.cpi.doc/ids_cpi_027.htm

If you are upgrading from an earlier OAT version, additional information is in the topic: http://www.ibm.com/support/knowledgecenter/SSGU8G_12.1.0/com.ibm.cpi.doc/ids_cpi_031.htm

Alternatively, the OpenAdmin Tool Community Edition is available from the IBM InformixIBM Informix Free Product Download page: https://www14.software.ibm.com/webapp/iwm/web/preLogin.do?lang=en_US&source=swg-informixfpd . The OpenAdmin Tool Community Edition includes OAT and the optional plug-ins, but it does not include the supporting software. This edition must be installed manually. Refer to the README file for installation instructions.
